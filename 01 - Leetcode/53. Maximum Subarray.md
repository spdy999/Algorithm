# 53. Maximum Subarray
---

## BF
General 2 loops [[2-pass loop]]

Time : `O(n^2)`; TLE
Space : `O(1)`

```python
maxx = 0
n = len(nums)
if n < 2:
	return nums[0]

for i in range(n): # O(n)
	prod = nums[i]
	maxx = max(maxx, prod)
	for j in range(i + 1, n): # O(n)
		prod *= nums[j]
		maxx = max(maxx, prod)
return maxx
```
