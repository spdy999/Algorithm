# 55. Jump Game
---

## [[Backtracking]]
Time : `O(2^n)`
Space : `O(n)`

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        # 2-pass
        n = len(nums)
        if n < 2:
            return True
        last_ind = n - 1

        def backtrack(i) -> bool:
            if i >= last_ind:
                return True
            k = nums[i]
            # for j in range(i + 1, i + k + 1):
            return any([backtrack(j) for j in range(i + 1, i + k + 1)])
         
        return backtrack(0)

```
